1. 定义: let set = new Set([1,2,3,4]);
  2. 转换为数组：var arr = [...set];
  3. set储存任何类型的唯一值，无论是原始值还是对象引用。

  5. set size设置
  • set.size() 返回set中值的个数
  • set.add(value) 在set对象尾部添加一个元素（如果set中没有），返回set对象
  • set.delete(0) 移除Set中与这个值相等的元素，返回Set.prototype.has(value)在这个操作前会返回的值。即如果该元素存在，返回true，否则返回false
  • set.has(0) 是否含有某元素
  • set.clear()
  • keys(): 返回一个新的迭代器对象，该对象包含Set对象中的按插入顺序排列的所有元素的值。
  • values():同keys()
  • entries() 返回键值对的遍历器 键值对的键和值是相等的。因为set无序且元素不可重复，所以没有索引一说

    • 这一方法是为了和Map对象保持一致。
